The following links showcase a few examples DNS domain integrations made possible by the ENS protocol.

## Coinbase Wallet ENS Integration (cb.id)

![Alt text](https://dnstoens.com/images/coinbase-logo1.png "Coinbase Wallet ENS Integration")

Coinbase has integrated the ENS protocol to issue cb.id names to their user base. For detailed information about this use case, see [DNS Off-Chain Subname Issuance - Coinbase Wallet (cb.id)](1-index)

## .ART TLD Registrar Integration with ENS

![Alt text](https://dnstoens.com/images/art-protocol-wht.png ".ART TLD integration with ENS")

The top-level domain registar for .ART domains has integrated their TLD infrastructure with the ENS protocol. For detailed information about this use case, see [Integrating an entire TLD with ENS - .ART Protocol](2-index)

## Mirror DNS app to an ENS dApp - ENS App Manager (app.ens.domains)

![Alt text](https://dnstoens.com/images/ens-logo-wht.png "ENS Logo")

The main registration and manager app for ENS names is available as a decentralized app using IPFS content. For detailed information about this use case, see [Mirror DNS app to an ENS dApp - ENS App Manager (app.ens.domains)](3-index)
