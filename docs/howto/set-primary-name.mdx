# Set a Primary name
:::caution Pre-requisites
 - The `ETH Address` record of the ENS name must first point to the wallet you want to set the `Primary Name` on

If you haven't set the `ETH Address` record to your ENS name, see
 - <Link to="/howto/add-wallet-address#eth-address"><b>How-To: </b>Add a Wallet Address <i>(ETH Address)</i></Link>
:::

Go to the <Link to="https://app.ens.domains"><b>ENS Manager App</b></Link> with the wallet set as `Manager` for the ENS name you want to manage and click `Names` to bring up a list
of your ENS names *or* search for an ENS name you own directly from the main page.
<BrowserWindow>
  <Image img="/img/howto/v3-set-primary-name-1.png"/>
</BrowserWindow>

Click the ENS name you want to set as `Primary` in your wallet.
<BrowserWindow>
  <Image img="/img/howto/v3-set-primary-name-2.png"/>
</BrowserWindow>

Click `Set as Primary name`
<BrowserWindow>
  <Image img="/img/howto/v3-set-primary-name-3.png"/>
</BrowserWindow>

:::tip
If your ENS name doesn't show up in the Primary drop-down list, it's usually because the ENS name's `ETH Address` record doesn't point to your ENS name. To resolve it, see
 - <Link to="/howto/add-wallet-address#eth-address"><b>How-To: </b>Add a Wallet Address</Link>
:::

## Verify that your Primary name works

Visit a few different services and verify that your `Primary name` is working correctly:

<Tabs>
<TabItem value="ENS Manager" default>

The [ENS Manager App](https://app.ens.domains) will show your `Primary name` and the `Avatar` set on it in the top-right corner, along with your wallet address.

<BrowserWindow url="https://app.ens.domains">
  <div style={{'background':'#f0f6fa'}}>
    <Image img="/img/howto/set-primary-name-verify-ensmanager.png"/>
  </div>
</BrowserWindow>

</TabItem>
<TabItem value="Uniswap">

[Uniswap](https://app.uniswap.org) will show your `Primary name` and its `Avatar` in the top-right corner as well!

<BrowserWindow url="https://app.uniswap.org">
<Image img="/img/howto/v3-set-avatar-services-uniswap.png"/>
</BrowserWindow>
</TabItem>
<TabItem value="OpenSea">

[OpenSea](https://opensea.io) shows your `Primary name` on your user's page next to your wallet address.
:::caution note
OpenSea's systems are _very_ slow to update to reflect any changes, please allow at least 24 hours for the `Primary name` to show.
:::
