Go to the <Link to="https://app.ens.domains"><b>ENS Manager App</b></Link> with the wallet set as `Manager` for the ENS name you want to manage and click `Names` to bring up a list
of your ENS names *or* search for an ENS name you own directly from the main page.
<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-1.png"/>
</BrowserWindow>

Click the ENS name you want to add records to.
<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-2.png"/>
</BrowserWindow>

Go to the `Records` tab
<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-3.png"/>
</BrowserWindow>

Click `Edit Records`
<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-4.png"/>
</BrowserWindow>

Click `Add Record`
<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-5.png"/>
</BrowserWindow>

In the new field that appears, type in the name of your Custom Record. Records traditionally use a reverse notation, e.g. `com.youtube` for youtube.com
and then click `Add`

<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-6-text-2.png"/>
</BrowserWindow>

Type in a value for your new record, and click `Save`
<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-6-text-3.png"/>
</BrowserWindow>

Then click `Open Wallet` and approve the transaction in your wallet.
<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-6-text-4.png"/>
</BrowserWindow>

Wait for the transaction to complete.
<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-6-text-5.png"/>
</BrowserWindow>

Once the transaction has completed your record is stored on-chain and should be immediately accessible.
<BrowserWindow url="app.ens.domains">
  <Image img="/img/howto/v3-set-record-6-text-6.png"/>
</BrowserWindow>
